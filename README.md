# ironNBA

Is a basic Node/Express backend application with handlebars templates, layout, partials.

[https://handlebarsjs.com/](https://handlebarsjs.com/)

## If you want to play with it

- Clone or fork it on your local
- Install packages
```
npm install
```
- start application
```
 node app.js (or nodemon app.js)
```

- open http://localhost:3000/
